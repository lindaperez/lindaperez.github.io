# Assets

Drop your images here. If a file is missing, the site automatically shows a
neutral gradient placeholder — nothing breaks.

## Files the site looks for

| File | Used for | Recommended size |
|------|----------|------------------|
| `profile.jpg` | Hero portrait | ~560×560 px (square) |
| `projects/project-1.jpg` … `project-6.jpg` | Project card thumbnails | ~800×450 px (16:9) |

## Tips
- Keep images optimized (compress JPGs/PNGs) for fast loading.
- Use descriptive filenames if you add more projects, and update the
  `PROJECTS` array in `../script.js` to point at them.
- Alt text is generated from each project title; edit titles in `script.js`.
